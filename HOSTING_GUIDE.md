# How to Host Your Jko Mobile Mechanic Website

## Quick Start: Easiest Options (Recommended for Beginners)

### Option 1: Netlify (Easiest - Drag & Drop) ⭐ RECOMMENDED

**Why:** Free, no coding required, takes 2 minutes

**Steps:**
1. Go to [netlify.com](https://www.netlify.com)
2. Click "Sign up" (use email or GitHub)
3. Once logged in, drag your entire folder (`untitled folder`) onto the Netlify dashboard
4. Wait 30 seconds - your site is live!
5. Netlify gives you a free URL like `jko-mobile-mechanic-123.netlify.app`
6. (Optional) Click "Site settings" → "Change site name" to customize the URL

**Cost:** FREE forever for basic hosting

---

### Option 2: GitHub Pages (Free, Professional)

**Why:** Free, reliable, good for learning

**Steps:**
1. Create a GitHub account at [github.com](https://github.com)
2. Install GitHub Desktop (easier) or use command line
3. Create a new repository called `jko-mobile-mechanic`
4. Upload all your files (index.html, css/, js/, assets/)
5. Go to repository Settings → Pages
6. Select "main" branch and click Save
7. Your site will be at `yourusername.github.io/jko-mobile-mechanic`

**Cost:** FREE

---

### Option 3: Vercel (Free, Fast)

**Why:** Very fast, free, easy setup

**Steps:**
1. Go to [vercel.com](https://vercel.com)
2. Sign up with GitHub or email
3. Click "Add New Project"
4. Drag and drop your folder or connect to GitHub
5. Deploy - done!

**Cost:** FREE

## Step-by-Step: Netlify (Recommended)

### Before You Start:
1. **Stop the local server** (if running):
   - In terminal, press `Ctrl+C` to stop the Python server

2. **Rename your folder** (optional but recommended):
   - Rename `untitled folder` to `jko-mobile-mechanic` or `website`

3. **Verify your files:**
   - Make sure you have: `index.html`, `css/`, `js/`, `assets/` folders

### Netlify Deployment:

1. **Sign Up:**
   - Visit [app.netlify.com](https://app.netlify.com)
   - Click "Sign up" → Use email or GitHub
   - Verify your email if needed

2. **Deploy:**
   - On the Netlify dashboard, find the area that says "Want to deploy a new site without connecting to Git? Drag and drop your site output folder here"
   - Drag your entire `jko-mobile-mechanic` folder onto that area
   - Wait for upload (30-60 seconds)

3. **Get Your URL:**
   - Netlify will show: "Your site is live at: `random-name-123.netlify.app`"
   - Click that URL to see your site!

4. **Customize Domain (Optional):**
   - Click "Site settings" → "Change site name"
   - Change to something like `jko-mobile-mechanic` (if available)
   - Your new URL: `jko-mobile-mechanic.netlify.app`

5. **Custom Domain (Later, if you buy one):**
   - If you buy `jkomobilemechanic.com` later, you can add it in Site settings → Domain management

## Troubleshooting:

**Problem:** Site looks broken after upload  
**Solution:** Make sure you uploaded the ENTIRE folder, not just index.html

**Problem:** CSS not loading  
**Solution:** Check that `css/` folder is in the same directory as `index.html`

**Problem:** Images not showing  
**Solution:** Make sure `assets/` folder was uploaded
